把更短的生活片段认真收好

一些平凡但闪光的日子

2026-03-18 GitHub Issues 驱动 微博流

微博流

终于明白为什么有人会说，博客是写给自己的情书了。

写给生活的页边注

为什么要记录微博

总有一些情绪太轻，轻到来不及写成一篇完整的文章；也总有一些瞬间太亮，亮到只想赶紧替它按下一个保存键。微博页就像给生活留出的窄窄书签，把那些一句话的心动、半日光阴的温柔、忽然想起的念头，静静夹在日子中间，等往后翻阅时，还能听见当时心里那一点点回声。

为什么要记录生活

认真记录生活，并不是为了把平凡过得多么隆重，而是想在忙碌和遗忘之间，替自己留下一点温热的证据。风从窗边吹过的时候、锅里冒起白气的时候、傍晚街灯亮起的时候，那些看似不起眼的小片段，往往才最像真正的日子。把它们写下来，像是在对未来的自己轻声说一句：你看，这一段路，我们也曾认真经过。
